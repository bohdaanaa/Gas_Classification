## ▶️ Інструкція до запуску проєкту

### 1. Встановлення інструментів

- Arduino IDE

### 2. Завантаження репозиторію

```bash
https://github.com/bohdaanaa/Gas_Classification
```

### 3. Імпорт бібліотеки

В Arduino IDE:
```bash
Sketch -> Include Library -> Add .ZIP Library
```
Gas_Classification_inferencing.zip

### 4. Запуск програми на ESP32

##### Відкрити файл `gas_classification.ino`;
##### Обрати плату ESP32 Dev Module та COM-порт;
##### Натиснути **Upload**.

### 5. Перегляд результатів

##### Відкрити **Serial Monitor** (Tools -> Serial Monitor);
##### Встановити швидкість 115200 baud.

## 🧪 Проблеми і рішення

| Проблема              | Рішення                            |
|----------------------|------------------------------------|
| Плата ESP32 не відображається в Arduino IDE | Перевірити USB-кабель, драйвери та правильність вибраного COM-порту |
| Помилка Failed to connect to ESP32: Wrong boot mode detected | Під час прошивки натиснути та утримувати кнопку BOOT на платі ESP32 |
| У Serial Monitor відображаються некоректні символи | Встановити швидкість порту 115200 baud |
